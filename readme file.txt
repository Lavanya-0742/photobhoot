# 📸 Long Distance Best Friends Photo Booth

A beautiful web-based photo booth experience designed especially for long-distance best friends. Capture memories together and create a realistic double photo strip just like a real photobooth.

---

## ✨ Features

- 📷 Camera permission using WebRTC
- 📸 Automatic capture of 3 photos
- ⏳ Animated countdown before each photo
- ⚡ Camera flash effect
- 👀 Photo preview
- 🔄 Retake photos anytime
- 🎞️ Generates two identical photo strips
- ✂️ Tear line between strips
- 📥 Download as high-quality PNG
- 📱 Responsive design
- 💖 Cute pastel Korean photobooth aesthetic

## 🚀 How to Run

### Method 1 (Recommended)

1. Install **Visual Studio Code**
2. Install the **Live Server** extension
3. Open the project folder
4. Right-click `index.html`
5. Click **Open with Live Server**

The project will automatically open in your browser.

---

## Camera Permission

When you click **Start Photo Booth**, the browser will ask for permission to use your webcam.

Click

```
Allow
```

The application cannot work without camera permission.

---

## 📸 Photo Booth Workflow

1. Open website
2. Click **Start Photo Booth**
3. Allow camera access
4. Countdown begins

```
3

2

1

📸
```

5. Three photos are captured automatically
6. Preview all photos
7. Retake if needed
8. Generate realistic photo strip
9. Download PNG

---

## 🎞️ Final Output

The website creates a realistic printable photo booth print.

```
┌──────────────┐   ✂   ┌──────────────┐

PHOTO 1             PHOTO 1

PHOTO 2             PHOTO 2

PHOTO 3             PHOTO 3

07 • 02 • 2026

FOREVER ❤️

└──────────────┘       └──────────────┘
```

The two strips are identical and can be cut apart after printing.
